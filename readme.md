# したみちRPG
## 制作アプリの説明
- モンスターと戦いレベルを上げ、魔王を倒そう！
- 勝利すると経験値獲得！レベルを上げて次のモンスターに挑もう！
- 負けるとステータスが減るぞ！かいふく や にげる を効果的に使おう！
- ひたすらモンスターと戦うだけなのでとても単調だぞ！がんばれ！
## 工夫した点・こだわった点
- localStorageを使用して冒険の書を管理
- 戦闘終了時に進捗をlocalStrageに保存(オートセーブ)
- モンスターオブジェクトはクラスを使用して生成
- autofocus、keyDownを使用し上下キーでコマンド選択
## 次回トライしたいこと（または機能）
- 複数モンスターとの戦闘
- 装備品・特殊技・魔法など、戦闘にバリエーションを持たせる
- 戦闘以外の部分の作成
